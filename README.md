## Функциональность
Передает текст на второй Activity по нажатию кнопки
По нажатию кнопки "Позвонить другу" открывается приложение для звонков
По нажатию кнопки "Поделиться текстом" появляется системное окно "Поделиться через..."
Данные валидируются
