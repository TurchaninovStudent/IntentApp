## Функциональность
Передает текст на второй Activity по нажатию кнопки<br>
По нажатию кнопки "Позвонить другу" открывается приложение для звонков<br>
По нажатию кнопки "Поделиться текстом" появляется системное окно "Поделиться через..."<br>
Данные валидируются
